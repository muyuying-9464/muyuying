JavaScript基础知识练习

1. 变量定义 (Variables)

测试 var、let、const 关键字的使用和作用域
// 变量定义示例 var globalVar = "全局变量"; let blockVar = "块级变量"; const constantVar = "常量";

2. 变量类型 (Data Types)

测试 JavaScript 的基本数据类型
// 数据类型示例 let number = 42; let string = "Hello"; let boolean = true; let array = [1, 2, 3]; let object = { name: "JavaScript" }; let func = function() { return "函数"; }; let undef = undefined; let nul = null;

3. 表达式 (Expressions)

测试算术、比较和逻辑表达式
// 表达式示例 let a = 10; let b = 5; // 算术表达式 a + b // 比较表达式 a > b // 逻辑表达式 a && b

4. 流程控制 (Control Flow)

测试 if-else、switch、循环等流程控制语句
// 流程控制示例 // if-else if (condition) { // 代码 } else { // 代码 } // switch switch (value) { case 1: // 代码 break; default: // 代码 } // 循环 for (let i = 0; i < 5; i++) { // 代码 }

5. 函数 (Functions)

测试函数定义、调用、参数和返回值
// 函数示例 // 函数声明 function add(a, b) { return a + b; } // 函数表达式 const subtract = function(a, b) { return a - b; }; // 箭头函数 const multiply = (a, b) => a * b;
